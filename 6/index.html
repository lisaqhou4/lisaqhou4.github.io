NeRF Framework Exploration

Introduction

Neural Radiance Fields (NeRF) revolutionize 3D scene reconstruction by representing scenes as volumetric fields parameterized by neural networks. However, traditional NeRF models struggle with dynamic scenes and require retraining for changes. Extensions like NeRF-W and D-NeRF address some limitations but lack efficient handling of multiple geometries within one model. Our project explores using conditional encodings to enable geometry variations, demonstrated with interactive outfit changes for a 3D figure.

Methodology

Dataset Preparation and Modification

We adapted the dataset from Instruct-NeRF2NeRF, editing images using InstructPix2Pix to create two subsets: original T-shirt images and modified suit images. This ensures consistency for NeRF training.

Model Set-up

The model introduces outfit and appearance embeddings to handle varying geometries and photometric features. Modified architecture incorporates these embeddings alongside positional encodings, enabling the representation of different geometries dynamically. A hierarchical coarse-to-fine approach ensures computational efficiency.
